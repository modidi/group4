## About
Aura Essence is a simple and interactive perfume e-commerce website that allows users to browse a collection of fragrances, search and filter products by category, add items to a shopping cart, adjust quantities, view the total cost and complete a basic checkout process. The website is built using HTML, CSS, and JavaScript, demonstrating JavaScript events, DOM manipulation, and responsive web design.
